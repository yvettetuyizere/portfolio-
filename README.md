# Yvette Tuyizere — Portfolio

A React + Vite portfolio site showcasing full-stack web and Flutter/Firebase mobile projects.

## Setup

1. Open a terminal in this folder.
2. Run `npm install`.
3. Start the site locally with `npm run dev`, then open the printed `http://localhost:5173` URL.
4. Build for deployment with `npm run build` (outputs to `dist/`).

## Files

- `package.json` — project dependencies and scripts
- `vite.config.js` — Vite React plugin configuration
- `src/main.jsx` — app entry point
- `src/App.jsx` — portfolio content and sections (about, skills, projects, experience, education, certificates, contact)
- `src/index.css` — portfolio styling

## Customize

- Swap the `YT` initials avatar in `src/App.jsx` for a real photo if you'd like.
- Add a LinkedIn link in the contact section once you have one.
- Add new projects to the `projects` array in `src/App.jsx` as you build them.
